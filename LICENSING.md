# RS41-NFW licensing

RS41-NFW is a mixed-license project. Most of it is GPL-3.0, one clearly marked
part is source-available (but not GPL), and a few bundled dependencies keep their
own upstream licenses. This document is the authoritative description of what is
covered by what.

If anything here and a source file disagree, the source file header wins for that
file, and this document wins for the overall structure.

## The GPL linking exception

The GPL-3.0 portion of RS41-NFW is offered under GPL-3.0 **with the following
additional permission**, granted by the author under section 7 of the GPL:

> As a special exception, the copyright holder of RS41-NFW gives permission to
> combine the GPL-3.0 licensed portions of RS41-NFW with the "Source-Available
> Modules" described in `LICENSE.source-available`, and to distribute the
> resulting combined firmware binary. You must obey the GNU GPL in all respects
> for all of the GPL-3.0 licensed code, as well as the RS41-NFW Source-Available
> License for the Source-Available Modules.

This exists so the two licenses can legally live in one compiled binary. Without
it, mixing GPL code with the more restrictive source-available modules in one
distributable binary would be a conflict.

## Source-Available Modules: the exact list

These functions are the sensor-boom acquisition/calibration, the sensor-boom and
humidity-module heating control, and the GPS-readout / NFW intelligent-GPS code.
They are covered by `LICENSE.source-available`, not the GPL. Each one is marked in
`rs41-nfw_sonde-firmware.ino` with an `// [RS41-NFW-SA]` banner immediately above
its definition (grep for `RS41-NFW-SA` to list them all).

Sensor boom:

- `selectSensorBoom`
- `getSensorBoomFreq`
- `getFactoryTc`, `getFactoryTH`, `getFactoryRH`
- `calibrateTempSensorBoom`
- `calculateSensorBoomResistance`
- `sensorBoomHandler`

Heating:

- `selectReferencesHeater`
- `flightHeatingHandler`
- `humidityModuleHeaterPowerControl`
- `extHeaterHandler`

GPS readout / intelligent GPS:

- `GPSManagement`
- `gpsHandler`
- `startGPS`, `shutdownGPS`, `restartGPS`, `initGPS`
- `gpsQuietMode`

Only the code between an `[RS41-NFW-SA]` marker and the end of the function it
introduces is source-available. Everything else in the file (including
`readThermistorTemp`) is GPL-3.0.
